Key Features

WiFi Analysis

Advanced scanning of access points and stations with OUI matching and real-time monitoring

Packet Capture

Capture probe requests, beacons, deauth packets, WPS, EAPOL and raw WiFi packets with PCAP support

Beacon Operations

Generate custom beacon frames with random, rickroll, or AP list modes for network testing

BLE Toolkit

Scan for Flipper devices, detect BLE spam, track AirTags, and capture raw BLE packets

Wardriving

GPS-enabled WiFi and BLE wardriving with CSV logging for network mapping

Evil Portal

Create customizable captive portals with online/offline modes and domain spoofing

Network Tools

DIAL/Chromecast control, network printer access, and TP-Link device management

Security Testing

WPS scanning, and skimmer detection capabilities and more

Display Features

Built-in games, music visualizer, and RGB rave mode for supported display boards

Getting Started

1

Flash Your Device

Visit the Espresso Flash Web Flasher

Bootloader Mode:

  1. Hold BOOT button
  2. Connect USB cable
  3. Release BOOT after connection
2

Basic Commands

  • scanap - Scan WiFi networks
  • list -a - List found networks
  • scanlocal - Scan local network
  • help - Show all commands
3

Touch Navigation

  • Top Half: Move Up
  • Bottom Half: Move Down
  • Middle: Select Item
  • Main Menu: Direct Touch
